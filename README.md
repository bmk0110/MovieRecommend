# 영화추천 웹서비스
영화 추천 웹 서비스
##Backend
$ cd final_api
$ pip install -r requirements.txt
$ python manage.py makemigrations
$ python manage.py migrate
$ python manage.py runserver

##Frontend
$ cd final_vue
$ npm install
$ npm run serve
